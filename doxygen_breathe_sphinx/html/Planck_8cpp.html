◆ compute_Planck_nuisance_prior_loglike()

void Gambit::CosmoBit::compute_Planck_nuisance_prior_loglike ( double &  result)

Apply Gaussian priors on some of the Planck nuisance parameters (cf. table 16 of 1907.1287) By default, the 2018 priors are used. If the 2015 priors should be considered, set "version: 2015" in the rules of this function. If needed, a custom file with the priors can be passed via the "prior_file" keyword in the rules.

Loop over all parameters and apply the prior

◆ compute_Planck_sz_prior()

void Gambit::CosmoBit::compute_Planck_sz_prior ( double &  result)

SZ- prior: Prior on the tSZ and kSZ amplitudes. Correlation is unconstrained by Planck. Use prior based on SPT and ACT data. (cf. https://arxiv.org/pdf/1507.02704.pdf – eq. 32)
