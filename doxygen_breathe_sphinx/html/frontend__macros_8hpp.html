Wrapping macros for backend-defined functions

BE_FUNCTION(NAME, TYPE, ARGSLIST, SYMBOLNAME, CAPABILITY, [(MODELS)], [REF]) is the macro used for constructing pointers to library functions and wrapping function pointers in backend functors.

The sixth argument (MODELS) is optional, and contains a list of models that you want this function to be able to be used with. The seventh argument REF is optional, and contains a string of citation keys as references for the backend function
