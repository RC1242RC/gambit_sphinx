137 // - SetNRCoefficient sets the coefficient of a single operator to a given value.
138 // This function can be called for WIMPs initialised via either SetWIMP_NREffectiveTheory or SetWIMP_NREFT_CPT, keeping in mind the different conventions
139 // Arguments are:
140 // (1) the WIMP index
141 // (2) The operator index, i.e. an integer specifying the non-relativistic operator, e.g. 6 for O_6, using the following numbering scheme:
142 // For WIMPs initialised via SetWIMP_NREffectiveTheory:
143 // i = 1, 3-15, 17 & 18 correspond to the standard non-relativistic operators from arXiv:1505.03117 (note that O_2 and O_16 are not included)
144 // i = -1: q^2 O_1
145 // i = -4: q^2 O_4
146 // For WIMPs initialisd via SetWIMP_NREFT_CPT:
147 // i = 1-12 correspond to the standard non-relativistic operators from arXiv:1203.3542 (see also arXiv:1708.02678, note that the operator O_2 is included)
148 // i = 13: 1/(mpi^2 + q^2) O_6
149 // i = 14: 1/(meta^2 + q^2) O_6
150 // i = 15: q^2/(mpi^2 + q^2) O_6
151 // i = 16: q^2/(meta^2 + q^2) O_6
152 // i = 17: 1/(mpi^2 + q^2) O_10
153 // i = 18: 1/(meta^2 + q^2) O_10
154 // i = 19: q^2/(mpi^2 + q^2) O_10
155 // i = 20: q^2/(meta^2 + q^2) O_10
156 // i = 21: 1/q^2 O_5
157 // i = 22: 1/q^2 O_6
158 // i = 23: 1/q^2 O_11
159 // i = 100: q^2 O_1
160 // i = 104: q^2 O_4
161 // (3) An isospin index: For WIMPs initialised via SetWIMP_NREffectiveTheory 0 stands for the isoscalar and 1 for the isovector component of the operator
162 // For WIMPs initialised via SetWIMP_NREFT_CPT 0 stands for the proton and 1 for the neutron component of the operator
163 // (4) The desired value of the operator coefficient in units GeV^-n, where n = 2 for the standard non-relativistic operators and n = 0, 2 or 4 for operators with explicit momentum pre-factors
